# UC5 - Exercícios de TypeScript

Exercícios práticos de TypeScript desenvolvidos como preparação para o Projeto Integrador (PI), cobrindo tipagem estática, interfaces, funções tipadas e tratamento de assincronismo.

## 📋 Exercícios

| # | Título | Arquivo | Status |
|---|--------|---------|--------|
| 1 | Modelagem Relacional e Manipulação de Listas | `src/exercise1.ts` | ✅ |
| 2 | Filtro de Busca Tipado | `src/exercise2.ts` | ✅ |
| 3 | Validação do Formulário de Cadastro | `src/exercise3.ts` | ✅ |
| 4 | Simulação de Banco de Dados Assíncrono | `src/exercise4.ts` | ✅ |

---

## 🛠️ Tecnologias

- **TypeScript**
- **[tsx](https://github.com/privatenumber/tsx)** para execução direta dos arquivos `.ts`

---

## 🚀 Como rodar

Instale as dependências do projeto:

```bash
npm install
```

Rode qualquer um dos exercícios executando o comando correspondente:

```bash
npx tsx src/exercise1.ts
npx tsx src/exercise2.ts
npx tsx src/exercise3.ts
npx tsx src/exercise4.ts
```

---

## 🌿 Fluxo de branches

- `main` — Versão estável (produção)
- `develop` — Integração das features
- `feature/*` — Uma branch por exercício, mesclada em `develop` via merge `--no-ff`
